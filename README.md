#### 软件架构

软件架构说明

#### 安装教程

1.  安装

```
yarn install
```

#### 使用说明

1.  调试

```
yarn dev
```

2.  测试

```
yarn test
```

3.  打包

```
yarn build
```

4.把 tsp 文件转换为 json 文件

```
 yarn tsp2json --inputdir=path/to/TSPLIB --outputdir=path/to/TSPLIB
```
